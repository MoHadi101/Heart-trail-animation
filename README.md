 Heart Trail Animation:
A beautiful and interactive heart trail animation that follows your cursor movement across the screen. Perfect for adding a romantic or playful touch to any website!

Features:
Interactive Heart Trail: Hearts follow your cursor as you move around the page

Smooth Animations: Elegant fade-out and scale effects

Randomized Properties: Each heart has unique size, rotation, and movement

Responsive Design: Works on all screen sizes and devices

Lightweight: Pure CSS and JavaScript - no dependencies required


 Demo:
Simply move your cursor around the screen to see the magical heart trail effect!

 Installation:
Clone or download this project folder

Open index.html in your web browser

That's it! Move your cursor to see the hearts
